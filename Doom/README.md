# Doom Mini Badge!

![Doom](doom1.png)

## Bill of Materials 
* 0.96" 80x160 ST7735 TFT LCD Display
* STMS001J3
* 22 Ohm 0603 Resistor
* 1uF 0603 Capacitor
* Pin headers


Same hardware as the Metroid Mini Badge

Doom face from orignal game. Shows random expression (always full health)
